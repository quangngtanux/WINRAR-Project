{{ if en }}

In this task, during {{ effort_duration }} you will be asked to adjust sliders to a specific target value. Select one of the {{ NUM_SLIDERS }} buttons and move the cursor to the target value. Once done click the Validate button. The button of the corresponding slider will turn orange to indicate that you have already adjusted it. Then select a new button and repeat the process.

{{ if subsession.treatment == INDIVIDUAL }}

Your earnings depend solely on your own performance in the task and do not depend on the performance of the other group members. For each correctly positioned slider, your gain will increase by {{ PIECE_RATE }} ECU. If the slider is incorrectly positioned, your gain will remain unchanged.

{{ elif subsession.treatment == COOPERATION }}

Your earnings in this task will depend on the best scorer in the group. The best scorer in the group is the group member who has correctly positioned the most sliders. For each slider that the best scorer completed, each group member will receive {{ PIECE_RATE }} ECU. Therefore, this means that the better your group members do on the task, the more you can earn.

{{ endif }}

You can do as many sliders as you want during the {{ effort_duration }} allocated to you.
You will be informed of your earnings for this task only at the end of the experiment.

{{ elif fr }}

Durant {{ effort_duration }} vous devrez positionner des curseurs sur une valeur cible donnée. Un total de {{ NUM_SLIDERS }} cibles vous sera proposé.

Vous devez d'abord sélectionner une valeur cible en cliquant sur un des {{ NUM_SLIDERS }} boutons à disposition. Une fois la valeur cible affichée, positionnez le curseur sur la cible donnée puis cliquez sur Valider. Le bouton de la cible correspondante devient alors orange pour vous indiquer que vous l’avez déjà positionnée. Choisissez ensuite une nouvelle cible et répétez le processus.

{{ if subsession.treatment == INDIVIDUAL }}

Vos gains dépendent uniquement de votre propre performance, ils ne dépendent pas de la performance des autres membres de votre groupe. Pour chaque curseur correctement positionné vous recevrez {{ PIECE_RATE }} ECU.

{{ elif subsession.treatment == COOPERATION }}

Vos gains dépendront du meilleur score dans le groupe. Le meilleur score dans le groupe est celui du membre du groupe qui a correctement positionné le plus de curseurs. Pour chaque curseur correctement positionné par le meilleur membre du groupe, chaque membre du groupe recevra {{ PIECE_RATE }} ECU. Ainsi, plus vos coéquipiers obtiennent de bons résultats dans cette tâche, plus vos gains seront élevés.

{{ endif }}

Vous pouvez positionner autant de curseurs que vous le souhaitez pendant les {{ effort_duration }} qui vous sont allouées.
Vous serez informé(e) de vos gains pour cette tâche à la fin de l'expérience.

{{ endif }}
