{{ if en }}

For this task, during {{ effort_duration }}, you will be asked to solve a series of addition problems. You will be offered {{ NUM_OPERATIONS }} operations. Each operation contains an addition problem. You first need to select one of the operations and then enter your response in the input box below the operation, then click the Validate button. The button of the corresponding operation will then turn orange to indicate that you have already solved it. Then select a new operation and repeat the process.

{{ if subsession.treatment == INDIVIDUAL }}

Your earnings depend solely on your own performance in the task and do not depend on the performance of the other group members. If the entered number is correct, your gain will increase by {{ PIECE_RATE }} ECU. If it is incorrect, your gain will remain unchanged.

{{ elif subsession.treatment == COOPERATION }}

Your earnings in this task will depend on the best scorer in the group. The best scorer in the group is the group member who has successfully completed the most operations. For each operation that the best scorer resolved, each group member will receive {{ PIECE_RATE }} ECU. Therefore, this means that the better your group members do on the task, the more you can earn.

{{ endif }}

You can do as many operations as you want during the {{ effort_duration }} that are allocated to you.
You will be informed of your earnings for this task only at the end of the experiment.

{{ elif fr }}

Durant {{ effort_duration }}, vous devrez calculer des sommes de nombres. Un total de {{ NUM_OPERATIONS }} opérations vous sera proposé. Chaque opération contient des nombres dont il faut calculer la somme.

Vous devez d’abord sélectionner une opération en cliquant sur un des {{ NUM_OPERATIONS }} boutons à disposition. Une fois l'opération affichée, calculez la somme des nombres, saisissez votre réponse dans la zone prévue à cet effet, puis cliquez sur Valider. Le bouton de l'opération correspondante devient alors orange pour vous indiquer que vous l’avez déjà calculée. Cliquez sur un autre bouton pour sélectionner une nouvelle opération et répétez le processus.

{{ if subsession.treatment == INDIVIDUAL }}

Vos gains dépendent uniquement de votre propre performance, ils ne dépendent pas de la performance des autres membres de votre groupe. Pour chaque opération correctement calculée, vous recevrez {{ PIECE_RATE }} ECU.

{{ elif subsession.treatment == COOPERATION }}

Vos gains dans cette tâche dépendront du meilleur score dans le groupe. Le meilleur score dans le groupe est celui du membre de votre groupe qui a réussi à calculer le plus d'opérations. Pour chaque opération correctement calculée par le meilleur membre du groupe, chaque membre du groupe recevra {{ PIECE_RATE }} ECU. Ainsi, plus vos coéquipiers obtiennent de bons résultats dans cette tâche, plus vos gains seront élevés.

{{ endif }}

Vous pouvez calculer autant d'opérations que vous le souhaitez pendant les {{ effort_duration }} qui vous sont allouées.
Vous serez informé(e) de vos gains pour cette tâche à la fin de l'expérience.

{{ endif }}
