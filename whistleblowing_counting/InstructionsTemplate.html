{{ if en }}

For this task, during {{ effort_duration }}, you will be asked to count 1’s in grids with {{ GRID_SIZE.0 }} rows and {{ GRID_SIZE.1 }} columns that contain only 0’s and 1’s. You will be offered {{ NUM_GRIDS }} grids.

You first need to select one of the grids and then count the number of 1’s in that grid. Once done, you need to enter the number in the input box below the grid and click the Validate button. The button of the counted grid will then turn orange to indicate that you have already counted it. Click on another button to select another grid and repeat the process.

{{ if subsession.treatment == INDIVIDUAL }}

Your earnings will depend solely on your own performance in the task and do not depend on the performance of the other group members. If the entered number is correct, your gain will increase by {{ PIECE_RATE }} ECU. If it is incorrect, your gain will remain unchanged.

{{ elif subsession.treatment == COOPERATION }}

Your earnings in this task will depend on the best scorer in the group. The best scorer in the group is the group member who has successfully completed the most grids. For each grid that the best scorer completed, each group member will receive {{ PIECE_RATE }} ECU. Therefore, this means that the better your group members do on the task, the more you can earn.

{{ endif }}

You can do as many grids as you want during the {{ effort_duration }} that are allocated to you.
You will be informed of you earnings for this task only at the end of the experiment.

{{ elif fr }}

Durant {{ effort_duration }}, vous devrez compter les "1" dans des grilles de {{ GRID_SIZE.0 }} lignes et {{ GRID_SIZE.1 }} colonnes composées uniquement de "0" et de "1". Un total de {{ NUM_GRIDS }} grilles vous sera proposé.

Vous devez d’abord sélectionner une grille en cliquant sur un des {{ NUM_GRIDS }} boutons à disposition. Une fois la grille affichée, comptez le nombre de "1" qu’elle contient, saisissez votre réponse dans la zone prévue à cet effet, puis cliquez sur Valider. Le bouton de la grille résolue devient alors orange pour vous indiquer que vous l’avez déjà comptée. Cliquez sur un autre bouton pour sélectionner une nouvelle grille et recommencez le processus.

{{ if subsession.treatment == INDIVIDUAL }}

Vos gains dépendent uniquement de votre propre performance, ils ne dépendent pas de la performance des autres membres de votre groupe. Pour chaque grille correctement comptée, vous recevrez {{ PIECE_RATE }} ECU.

{{ elif subsession.treatment == COOPERATION }}

Vos gains dans cette tâche dépendront du meilleur score dans le groupe. Le meilleur score est celui du membre du groupe qui aura correctement compté le nombre de "1" dans le plus de grilles. Pour chaque grille correctement comptée par le meilleur membre du groupe, chaque membre du groupe recevra {{ PIECE_RATE }} ECU. Ainsi, plus vos coéquipiers obtiennent de bons résultats, plus vos gains seront élevés.

{{ endif }}

Vous pouvez compléter autant de grilles que vous le souhaitez pendant les {{ effort_duration }} qui vous sont allouées.
Vous serez informé(e) de vos gains pour cette tâche à la fin de l'expérience.

{{ endif }}
